<br>
realiza un salto de línea. 

 <p> </p>
engloba un párrafo. 

 <b> </b> y <strong> </strong>
ponen el texto en negrita. La diferencia entre ellas, es que strong es una etiqueta semántica e indica al navegador que el texto que hay dentro es realmente importante. 

 <u> </u> e <ins> </ins>  s>
subrayan un texto. Ins es una etiqueta semántica. 

 <i> </i> y <em> </em>
ponen el texto en cursiva. Em es semántica. 

 <h1> </h1> ... <h6> </h6>
son etiquetas para títulos. Cuanto más pequeño es el número, más importancia tiene para el navegador. 

 <small> </small>
hace más pequeño el tamaño de un texto. 

 <big> </big>
hace más grande el tamaño de un texto. 

 <sub> </sub>
hace que un texto se convierta en un subíndice. 

 <sup> </sup>
hace que un texto se convierta en un superíndice. 

 <strike> </strike> o <del> </del>
tachan un texto. Strike es semántica 

 <pre> </pre>
muestra el código dentro de una página, es decir, el navegador no interpreta el código que hay dentro y muestra el código tal cual. 

 <abbr> </abbr>
representa una abreviación en siglas del título que le indiquemos en la misma etiqueta. 

 <mark> </mark>
resalta el texto. 

 <a> </a>
crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL. 

 <img />
añade imágenes a través de su ruta que le marcamos con el tributo src. No tiene etiqueta de cierre. 

 <ol> </ol> y <ul> </ul>
crean listas. Ol para hacer una lista ordenada y ul para una desordenada. 
 <li> </li>
añade elementos a dichas tablas. 

 <blockquote> </blockquote>
crea citas en bloque, marca las citas a otros autores o documentos.

La etiqueta que te permite ver el código preformateado es la etiqueta pre.
